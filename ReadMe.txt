# User manual 

## Save 
* saves into your directory and converted into pdf 

## Generate
* generates a random worksheet based on what was inputted

## Number of problems
 the number of problems the user wants on their worksheet (max 51)

## Min, max
*  limits the problems to only show numbers in a certain range (min - nonnegative integer, max - integer greater than min)
- ex. 1 as min and 5 as max would make the worksheet only show problems that have numbers from 1 to 5

## Check boxes
*  check at least one of the operations and it will include those operations on the worksheet

## Show answer
* when you generate the worksheet, it will show the answer sheet with it if you press this button
